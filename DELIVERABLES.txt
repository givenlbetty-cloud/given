🔐 SÉCURITÉ IMPLÉMENTÉE
═══════════════════════════════════════════════════════════════════════════════

Protection 1: ACCÈS PROTÉGÉ
   ├─ Basic Auth Middleware
   ├─ Demande identifiants avant accès
   ├─ Configurable via ENABLE_BASIC_AUTH=True
   └─ Rollback possible en 30 sec

Protection 2: CONFIGURATION ISOLÉE
   ├─ settings_beta.py entièrement séparé
   ├─ Ne partage aucun paramètre avec production
   ├─ Chargement de .env.beta (pas .env)
   └─ SECRET_KEY unique générée

Protection 3: BD ISOLÉE
   ├─ db_beta.sqlite3 vs db.sqlite3
   ├─ Zéro synchronisation
   ├─ Données de test complètement séparées
   └─ Rollback restaure l'original

Protection 4: MÉDIAS SÉPARÉS
   ├─ media_beta/ vs media/
   ├─ Uploads de test ne touchent pas production
   ├─ Nettoyable sans impact
   └─ Facilite l'archivage après tests

Protection 5: SERVICES TIERS TEST
   ├─ Stripe clés TEST uniquement (pk_test_, sk_test_)
   ├─ Validation automatique dans settings_beta.py
   ├─ Email via console (pas d'envoi réel)
   ├─ Zéro coût, zéro impact réel
   └─ Bac à sable complet

Protection 6: DOMAINE ISOLÉ
   ├─ ALLOWED_HOSTS = domaine beta seulement
   ├─ Autres domaines = 400 Bad Request
   ├─ CSRF_TRUSTED_ORIGINS = origines beta
   └─ Impossibilité d'accès depuis production


🚀 DÉPLOIEMENT RAPIDE (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

Étape 1: Préparer (1 min)
   $ cp .env.beta.example .env.beta
   $ nano .env.beta
   ✏️  Remplir: SECRET_KEY, BASIC_AUTH_PASSWORD, domaine, Stripe TEST

Étape 2: Valider (1 min)
   $ python3 validate_security.py
   ✅ Doit afficher: "ENVIRONNEMENT BETA SÉCURISÉ"

Étape 3: Déployer (2 min)
   $ bash deploy_beta.sh
   ✅ Crée db_beta.sqlite3, media_beta/, staticfiles_beta/

Étape 4: Lancer (instant)
   $ bash run_beta.sh
   ✅ Serveur sur http://localhost:8000

Étape 5: Accéder (instant)
   🌐 http://localhost:8000
   🔐 basic_user / [password]
   🔑 /admin/ - admin_beta / beta_admin_password_123

🔄 PROCÉDURES D'URGENCE
═══════════════════════════════════════════════════════════════════════════════

La production est affectée?

  ├─ Arrête le serveur
  ├─ Revient à main branch
  ├─ Restaure la BD originale
  ├─ Relance le serveur
  └─ Temps: < 30 secondes

  $ bash deploy_beta.sh
  └─ Recrée une BD vierge

Serveur ne démarre pas?
  $ python3 manage.py check --settings=atj_site.settings_beta
  $ cat logs/gunicorn_error.log
